Fungsinya haruslah tidak memodifikasi arraynya. Itu haruslah mengembalikan array baru.

Contoh:

```js
let arr = [5, 3, 8, 1];

let filtered = filterRange(arr, 1, 4); 

alert( filtered ); // 3,1 (mencocokan nilai)

alert( arr ); // 5,3,8,1 (tidak diubah)
```

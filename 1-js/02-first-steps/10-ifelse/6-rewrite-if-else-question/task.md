# Tulis ulang 'if..else' menjadi '?'

Tulis ulang `if..else` menggunakan beberapa operator ternary` '?' `.

Agar mudah dibaca, disarankan untuk membagi kode menjadi beberapa baris.

```js
let message;

if (login == 'Employee') {
  message = 'Hello';
} else if (login == 'Director') {
  message = 'Greetings';
} else if (login == '') {
  message = 'No login';
} else {
  message = '';
}
```
